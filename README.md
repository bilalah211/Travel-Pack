# Travel Pack App ✈️🎒

Travel Pack App is an Android application designed to help travelers organize their trips by generating
smart packing lists based on destination, weather conditions, and trip details.

The app uses **Gemini AI** to create personalized packing suggestions and integrates a **Weather API** to 
adjust recommendations according to real-time weather data. All user data and packing lists are securely 
stored using **Firebase**.

---

## 🚀 Features

- 🤖 **AI Packing List Generator**  
  Uses **Gemini AI** to generate personalized packing lists based on trip details.

- 🌤 **Weather-Based Recommendations**  
  Integrates a **Weather API** to analyze the destination weather and suggest appropriate items.

- 🔥 **Firebase Integration**  
  Stores trip details, packing lists, and images securely in Firebase.

- 🖼 **Image Upload Support**  
  Users can upload images related to packing items or their trips.

- 📝 **Editable Packing Lists**  
  Users can modify AI-generated packing lists according to their needs.

- 📱 **User-Friendly Interface**  
  Simple and modern Android UI for easy travel planning.

---

## 🛠 Tech Stack

- **Kotlin** – Android development
- **Gemini AI API** – AI-based packing list generation
- **Weather API** – Real-time weather information
- **Firebase** – Cloud database and storage
- **Android Studio** – Development environment

---

## 📦 How It Works

1. User enters trip details such as destination and trip duration.
2. The app retrieves weather data using the Weather API.
3. **Gemini AI** generates a smart packing list.
4. Users can edit the list and upload images if needed.
5. All data is saved in **Firebase** for easy access.
